# Шрифты Loom Store (из макета Figma)

Шрифты подключены из подпапок:

| Папка         | Файлы                    | Использование        |
|---------------|--------------------------|----------------------|
| `charito/`    | Charito.ttf              | Заголовки, «Loom Store» |
| `etude_noire/`| Etude Noire Regular.ttf  | Пилсы категорий      |
| `akony/`      | AKONY.woff2, .woff, .ttf | Логотип в шапке      |

**Montserrat** подключается через Google Fonts в приложении.
